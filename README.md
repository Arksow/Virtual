# Memory Sequence

## Run the shared game server

1. Install Node.js 20 or later.
2. Run `npm install` in this folder.
3. Run `npm run build`.
4. Run `npm start`.

Open `http://localhost:5173` on the host computer. To join from phones, connect every device to the same Wi-Fi and open `http://<host-computer-LAN-IP>:5173`.

Create a room on one phone and share its four-letter code. Up to four additional phones can join.
