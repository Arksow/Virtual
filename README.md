# Memory Sequence — ADRENALIN Blink Key

Each row lists the number of blinks in this order: **Green · Blue · Pink · Orange**.

| Position | Letter | Blink counts |
| --- | --- | --- |
| 1 | A | 1 · 1 · 1 · 2 |
| 2 | D | 4 · 1 · 2 · 1 |
| 3 | R | 3 · 4 · 5 · 3 |
| 4 | E | 5 · 1 · 4 · 4 |
| 5 | N | 4 · 3 · 2 · 1 |
| 6 | A | 1 · 1 · 1 · 2 |
| 7 | L | 2 · 3 · 3 · 5 |
| 8 | I | 4 · 2 · 2 · 1 |
| 9 | N | 4 · 3 · 2 · 1 |

The game randomizes only the order in which colours blink. The totals above stay the same, so the decoder can match each observed pattern to a letter.
